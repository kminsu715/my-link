# My Link (Profile Project)

![Next.js](https://img.shields.io/badge/Next.js-16-black?style=for-the-badge&logo=next.js)
![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)
![Tailwind CSS](https://img.shields.io/badge/Tailwind--CSS-v4-38B2AC?style=for-the-badge&logo=tailwind-css)

모던 포트폴리오 및 링크트리 웹 애플리케이션 프로젝트입니다. 이 레포지토리는 전체 워크스페이스를 담고 있으며, 핵심 코드는 `my-profile` 폴더에 위치해 있습니다.

## 🚀 기술 스택 (Tech Stack)
- **Framework:** Next.js 16 (App Router)
- **Library:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4, Glassmorphism UI
- **Deployment:** Vercel (예정)

## 📁 프로젝트 구조 (Repository Structure)
- `/my-profile`: 핵심 서비스를 담당하는 Next.js 메인 패키지입니다.
- `/GEMINI.md`: AI 개발을 위한 컨텍스트 및 프롬프트 가이드 문서입니다.

## 💻 로컬 실행 방법 (How to run)

```bash
cd my-profile
npm install
npm run dev
```
로컬 개발 서버가 실행되면 브라우저에서 `http://localhost:3000` 으로 접속하여 프로필을 확인할 수 있습니다.
